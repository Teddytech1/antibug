Default Display Names

The default display name for a test method is the name of the method concatenated with a comma-separated list of parameter types in parentheses. The names of parameter types are retrieved using . For example, the default display name for the following test method is testUser(TestInfo, User). 

   @Test
   void testUser(TestInfo testInfo, @Mock User user) { ... }
