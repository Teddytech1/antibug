ThreadSpecific

This lifestyle guarantees that only one instance of a specific class exists per thread.

This class can be used instead of class like the following: 

 private static final ThreadSpecificSimpleDateFormat formatter = new ThreadSpecific(SimpleDateFormat.class);

This is equivalent to the next code. 

 private static final ThreadLocalSimpleDateFormat formatter = new ThreadLocalSimpleDateFormat() {
     Override
     protected SimpleDateFormat initialValue() {
         return new SimpleDateFormat();
     }
 };
